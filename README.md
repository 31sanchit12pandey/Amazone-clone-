# Amazon Clone

This project is a simple front-end clone of the Amazon homepage built using HTML and CSS.

## Features
- Responsive layout
- Product cards and sections
- Styled header, navbar, and hero area
- Image-based Amazon-style UI

## Files
- `finalclone.html` - Main HTML structure
- `amazonclone.css` - Styling for the page

## Preview
Open `finalclone.html` in your browser to view the project.
